About The Platform

Finance è una piattaforma a scopo simulativo e didattico, realizzata come esercitazione atta all'integrazione di più tecnologie. Il server è hostato su una macchina virtuale a kernel linux, con sistema operativo Clear OS. Il database che ospita i vari dati è semplicemente un database MariaDb con motore mysql, l'accesso ad esso è consentito solo ad alcune utenze adeguatamente configurate. La macchina virtuale,inoltre supporta diversi servizi, come FTP,SAMBA,Flexshare e server SMP (non ancora testato).
